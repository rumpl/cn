# Chuck Norris jokes

Simple cli client for the [icndb](http://www.icndb.com/) database.

## Installation

`npm install cn -g`

## Usage

Just execute `cn`:

```
$ cn
Chuck Norris doesn't believe in ravioli. He stuffs a live turtle with beef and smothers it in pig's blood.
```
